SULinux

SUWAS

Web Server / Web Application Server 솔루션입니다.

Apache / Tomcat기반의 WAS솔루션으로 18년 동안 도입 및 기술지원하고 있습니다.
타 사용 WAS에 비해 안정성과 성능이 뛰어나고, 확장성이 좋고 관리가 편리합니다.

Serlet/JSP컨테이너와 OpenEJB등을 지원합니다.
NIPA, KTNET, KRX등 주요시스템에 80여개의 도입실적이 있습니다.

SUCLUSTER

HA구성을 위한 클러스터시스템 솔루션입니다.

전산시스템 장애에 실시간으로 대응하는 무중단 시스템을 위한 솔루션입니다.
Fencing / 리소스관리 / 공유파일시스템(GFS2) 지원/ 실시간볼륨복제기술 / 공유스토리지 볼륨매니저등의 특징을 가지고 있습니다. GUI관리 인터페이스 지원합니다.

근로복지공단, 부산광역시, 부천대학교, 성경방송, 외교부, 국립세종도서관, 조선비치호텔등 100여 도입 실적이 있습니다.
